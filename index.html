What this project does

Built for a real society delivery business. Every piece works in production.

🤖
AI-Powered WhatsApp Bot
Customers chat naturally in English or Hinglish. The bot loads their full profile, order history, and cart from the database before every message — so it can answer "what did I order last time?" or "what's my address?" from real data.
📊
Real-Time Dashboard
Secure login-protected web dashboard. View live orders, update delivery status, manage products, see customer database, handle support requests, send broadcast messages, export CSV.
🧠
Context-Aware Intelligence
The LLM receives the customer's complete Firebase context on every message — profile, past orders with statuses, current cart, live menu, active deals. It answers questions from real data, not scripts.
🏗️
Production Architecture
Node.js + Express server deployed on Render. Firebase Firestore database (Mumbai region). Twilio WhatsApp API. Groq LLM inference. Razorpay payments. UptimeRobot monitoring. Full CI/CD via GitHub.
💬
Full Conversation Flow
State-managed conversations with profile collection, product catalog browsing, cart management, address confirmation, payment selection, and order confirmation. Global commands work from any state.
₹0
Zero-Budget Trial Stack
Entire system runs for ₹0/month during trial. Groq free tier (14,400 req/day), Firebase Spark plan, Render free hosting, Twilio sandbox. Clear upgrade path to production as revenue grows.

Run it yourself

Clone and run locally in under 5 minutes

# Clone the demo repo git clone https://github.com/SaransGoel/breaddash-demo.git cd breaddash-demo # Install dependencies npm install # Configure (optional - demo works without real keys) cp .env.example .env # Add your Groq key for live AI: https://console.groq.com # Start server npm start # Open dashboard open http://localhost:3000/dashboard # Login: admin / breaddash123
# Deploy to Render (free) # 1. Fork this repo to your GitHub # 2. Go to render.com → New Web Service # 3. Connect your fork # 4. Settings: # Build: npm install # Start: npm start # 5. Add environment variable: # GROQ_API_KEY = your_key_from_console.groq.com # 6. Deploy → get your live URL # # For full production with real WhatsApp + Firebase: # See the private repo README
